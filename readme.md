# 🚗 FCSM Covoiturage

Plateforme de covoiturage dédiée aux supporters du FC Sochaux-Montbéliard pour faciliter les déplacements vers le stade les jours de match. Et proposer une alternatif avec Json pour les groupes

---

## 🌐 Accès au site

👉 https://magenta-dogfish-494017.hostingersite.com/index.php

---

## ⚙️ Fonctionnalités principales

### 1️⃣ Proposer un trajet
Les conducteurs peuvent publier une offre de covoiturage en renseignant :
- leur véhicule
- le nombre de places
- le trajet et les horaires

---

### 2️⃣ Réserver un trajet
Les passagers peuvent :
- rechercher un trajet disponible
- faire une demande de réservation
- suivre l’état de leur demande

---

### 3️⃣ Messagerie intégrée
Un système de chat permet :
- aux passagers et conducteurs de discuter
- d’échanger des informations sur le trajet

---

## 🛠️ Technologies utilisées

- PHP
- MySQL
- HTML / CSS
- JavaScript
- API OpenStreetMap (géolocalisation)

---

## 📌 Objectif du projet

Simplifier l’organisation des déplacements des supporters en centralisant :
- les offres de trajets
- les demandes
- la communication entre utilisateurs
